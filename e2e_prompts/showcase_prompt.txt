<structure>
Документ ОБЯЗАН содержать ВСЕ следующие элементы — это showcase возможностей MINT:

1. ОБЛОЖКА — название документа (крупно, по центру), компания, автор, дата, декоративная линия
2. ОГЛАВЛЕНИЕ — TableOfContents с headingStyleRange "1-3"
3. МИНИМУМ 3 СЕКЦИИ с Heading 1, внутри — Heading 2 и текст по теме {{TOPIC}}
4. ТАБЛИЦА — минимум 4 колонки, header с заливкой, zebra striping, ширины в DXA
5. СПИСОК маркированный (2+ уровня вложенности) через numbering config
6. СПИСОК нумерованный (2+ уровня)
7. CALLOUT-БЛОК — параграф с левой рамкой (12pt, синяя), заливкой фона, иконкой-текстом
8. WARNING-БЛОК — то же, но жёлтая рамка + иконка предупреждения
9. CODE-БЛОК — Courier New на сером фоне (#F5F5F5) с рамкой
10. КОЛОНТИТУЛЫ — header: название документа + правый tab stop с датой; footer: компания + "Стр. X из Y"
11. НУМЕРАЦИЯ СТРАНИЦ через PageNumber.CURRENT / TOTAL_PAGES
12. HYPERLINK — хотя бы одна внешняя ссылка
13. ФОРМАТИРОВАНИЕ СИМВОЛОВ — продемонстрировать bold, italic, underline, highlight, color, superscript в одном абзаце
</structure>

<critical_rules>
ОБЯЗАТЕЛЬНЫЕ правила — нарушение = невалидный документ:

- Размер страницы: US Letter (width: 12240, height: 15840 DXA), margins: 1440
- Ширина контента: 9360 DXA (12240 - 2x1440)
- Таблицы: ВСЕГДА WidthType.DXA (НИКОГДА PERCENTAGE), сумма columnWidths = width таблицы, width на каждой ячейке = её columnWidth
- Заливка ячеек: ТОЛЬКО ShadingType.CLEAR (SOLID = чёрный фон)
- Cell margins: { top: 80, bottom: 80, left: 120, right: 120 }
- Списки: ТОЛЬКО через numbering config + LevelFormat.BULLET/DECIMAL. ЗАПРЕЩЕНО: TextRun с символами буллетов
- Стили заголовков: ТОЧНЫЕ ID "Heading1", "Heading2", "Heading3" с outlineLevel 0, 1, 2
- Переносы строк: ЗАПРЕЩЕНО "\n" — каждая строка = отдельный Paragraph
- PageBreak: ТОЛЬКО внутри Paragraph
- Footers: ЗАПРЕЩЕНЫ таблицы. Используй PositionalTab для двух-колоночной раскладки
- Sizes: TextRun size в half-points (24 = 12pt)
- Spacing: в DXA (20 DXA = 1pt, 1440 DXA = 1 inch)
</critical_rules>
